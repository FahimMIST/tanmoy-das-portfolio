# Tanmoy Das - Professional Portfolio

This is a modern, responsive professional portfolio website for Tanmoy Das, an Operations Research Scientist and Data Scientist. It showcases professional experience, academic background, technical skills, and featured projects.

Built with React, TypeScript, Tailwind CSS, and Vite.

## ğŸ“¦ Getting Started

### Prerequisites

Ensure you have Node.js (version 18+ recommended) installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd tanmoy-das-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
